# 📈 Luxuryverce AI Big-Data Finance Engine

An intelligent, production-grade automated personal finance dashboard that processes large-scale banking statements locally via Pandas vectorized analytics and generates executive strategic summaries using the `gpt-4o-mini` LLM engine.

## 🚀 Key Features

- **Big-Data Optimization:** Built with a rule-based localized pre-categorization algorithm capable of sorting 10,000+ data lines under a millisecond without incurring API lag or cost overheads.
- **Smart Data Chunking:** Compresses raw transactional registers into numerical frequency tables (`groupby` maps) before transmission to protect prompt context window capacities.
- **Robust Currency Sanitizer:** Utilizes an integrated heavy-duty regex scanner to auto-clean multi-currency artifacts, stray formatting symbols (`?`, `,`), and custom spacing layouts.
- **Multi-Column Auto-Detection:** Intelligently maps independent `Withdrawal` vs `Deposit` multi-boundary layouts or traditional single unified `Amount` lines.
- **Interactive Visualizations:** Renders dynamic live cash balance timeline histories and expense volume trends via Plotly Express.

---

## 🛠️ Architecture & Stack

- **Frontend Interface:** Streamlit (Wide-screen responsive deployment layout)
- **Data Engineering:** Pandas Engine (Vector calculations & dynamic matrix filters)
- **Data Visualization:** Plotly Express (Interactive Line & Bar charts)
- **AI Core Pipeline:** OpenAI API Wrapper (`gpt-4o-mini` via production endpoint model grids)
